Perguntas Frequentes (FAQ) Git:

Como configuro meu usuário?
Use git config --global user.name "Seu Nome" e git config --global user.email "seuemail@exemplo.com".
Como corrigir o último commit?
Use git commit --amend para alterar o último commit, permitindo adicionar arquivos esquecidos ou editar a mensagem.
Para que serve a "staging area" (git add)?
Serve para preparar um "snapshot" (instantâneo) das alterações antes de registrá-las no histórico, permitindo maior controle sobre o que é incluído no commit.
Como desfazer um git add?
Use git reset para remover arquivos da área de stage sem perder as alterações feitas.
Como ignorar arquivos/pastas?
Crie um arquivo .gitignore na raiz do projeto e adicione os nomes dos arquivos ou padrões que deseja ignorar.
O que fazer em erro de "non-fast-forward"?
Isso ocorre quando o repositório remoto está mais atualizado que o seu local. Use git pull para mesclar as alterações remotas antes de fazer o git push.
Como criar e mudar para uma nova branch?
Use git checkout -b "nome-da-branch".
